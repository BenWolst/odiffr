# Clear the odiffr Cache

Removes all cached binaries downloaded by
[`odiffr_update()`](https://benwolst.github.io/odiffr/reference/odiffr_update.md).

## Usage

``` r
odiffr_clear_cache()
```

## Value

Invisibly returns `TRUE` if successful, `FALSE` otherwise.

## Examples

``` r
if (FALSE) { # \dontrun{
odiffr_clear_cache()
} # }
```
